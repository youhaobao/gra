# 远程网络监控系统UI重构计划

## 1. 布局结构修改
- **修改mainwidget.ui**：将当前的网格布局改为三栏式布局
  - 左侧：约20%宽度，垂直排列天气卡片和音频监控卡片
  - 中间：约60%宽度，主视频监控窗口
  - 右侧：约20%宽度，垂直排列功能按钮组
- **设置整体宽高比**：调整为16:9或5:3，适合1024×600或1280×720分辨率

## 2. 视觉风格实现
- **更新样式表**：
  - 背景：浅天蓝色 #E6F2FF
  - 卡片/面板：白色 #FFFFFF，带8-12px圆角和轻微阴影
  - 强调色：亮蓝色 #1677FF（用于选中按钮、进度条等）
  - 文字颜色：深灰 #333333（主文字），浅灰 #999999（次要文字）
- **添加阴影效果**：使用QGraphicsDropShadowEffect提升层次感

## 3. 模块改造
### 左侧信息卡片
- **天气卡片**：
  - 顶部：太阳和云雨图标
  - 中部：温度和时间数据
  - 底部：虚线分隔的装饰
- **音频监控卡片**：
  - 顶部：波形图（保留现有实现）
  - 中部：三个水平进度条（表示不同音频参数）
  - 进度条样式：蓝色填充，浅灰背景

### 中间视频监控区
- **视频显示区域**：使用QVideoWidget或QLabel显示帧
- **底部操作按钮**：截图、拍照、文件/回放
  - 按钮样式：圆角矩形，白色背景，图标居中
  - 选中态：蓝色背景 + 白色图标
  - 未选中态：白色背景 + 深色图标

### 右侧功能按钮区
- **顶部水平图标按钮**：设置、语音、截图、文件、AI分析
- **中部垂直图标按钮**：锁定、云台控制、设置
  - 按钮样式：白色背景，圆角矩形，图标居中
  - 选中态：蓝色边框或背景高亮

## 4. 技术实现要点
- **布局管理**：使用QHBoxLayout做整体三栏布局，QVBoxLayout做左侧卡片堆叠和右侧按钮堆叠
- **样式表**：统一设置按钮、卡片、进度条的样式
- **自定义绘制**：波形图使用QPainter绘制，添加虚线装饰
- **图标**：使用线性图标，风格统一简洁

## 5. 具体文件修改
1. **mainwidget.ui**：重新设计布局结构
2. **mainwidget.cpp**：更新样式表和布局逻辑
3. **audiowidget.cpp**：添加进度条功能
4. **WeatherClockWidget.cpp**：调整为卡片样式
5. **videowidget.cpp**：确保视频显示区域适配新布局
